Project Description

iStitch is a image processing web application for corneal nerve quantification. It stitches images and videos into montages using the OpenCV stitching algorithm. It then filters the montages to isolate nerves and calculate statistics. Statistics include: corneal nerve fiber tortuosity (CNFT), corneal nerve branch density (CNBD), corneal nerve fiber density (CNFD) and corneal nerve fiber length (CNFL).

Project Details

  • iStitch is currently hosted at istitch.github.io.
  • iStitch was made to help distinguish significant patterns and disorders in a patient's cornea which would help diagnose diseases like type 2 diabetes.
  • I created the website for iStitch which allows users to easily process a video of a patient's cornea into a stitched image.
  • I worked with Yash Vaishnav, an aspiring doctor, to create this application to demonstrate his research in opthalmology
