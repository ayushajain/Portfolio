Project Description

Dexto is a revolutionary product that translate electromyographic signals into readable data to be used for prosthetics or robotics. Our web interface clearly models movements in the user's arm and all of the data can be accessed through our free API. We used a Myo to get raw data from its EMG sensors and displayed this in an easy to use interface that displays the users hand movements in a 3D model. This product was created at Angel Hack 2015 and won the award for "best use of Myo."

Project Details

  • Dexto was made at Angel Hack 2015 and is currently hosted at dextolabs.github.io
  • Our product allows users to easily translate overwhelming EMG data into simple angles and coordinates for use in Robotics and Prosthetics
  • Currently we are working on a neural network algorithm for users to create their own myo gestures with ease
  • Dexto will soon host 1 more sub-application: Sign language for those with hearing imparities
